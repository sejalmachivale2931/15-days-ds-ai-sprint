# 15-Day Data Science & AI Practice Sprint

I already understood the core concepts — Python, ML, DL basics — but was missing hands-on practice. This repo documents 15 days of daily, hands-on data science and AI practice: real datasets, real code, real mistakes fixed.

## 🎯 Goal
By Day 15: 2 solid, well-documented projects + a clear grasp of core ML/DL/GenAI concepts + interview-ready confidence.

## 🗺️ Plan

**Week 1 — Python + Machine Learning**
- Advanced Python (OOP, decorators, generators)
- SQL + data wrangling
- EDA, feature engineering
- Model building & comparison (Regression, Random Forest, XGBoost)
- Hyperparameter tuning

**Week 2 — Deep Learning + GenAI**
- Neural networks with PyTorch
- Transformers & attention mechanism
- GenAI: prompt engineering, RAG pipelines
- Portfolio polish + interview prep
